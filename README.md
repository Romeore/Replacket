## Introduction

Replacket is an open-source.NET-based application that replays previously captured network traffic with specific or real arrival time deltas between packets.

## System Requirements

- Windows 7 and above
- .NET Framework 4.5.2

## Technologies

- WPF
- MaterialDesign
- MvvmLight
- SharpPcap
